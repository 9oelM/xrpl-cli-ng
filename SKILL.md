## Installation

**Requirements:** Node.js 22 or higher.

```bash
# Global install (recommended)
npm install -g xrpl-cli-ng

# Zero-install alternative (no global install required)
npx xrpl-cli-ng <command>
```

Smoke-test after install:

```bash
xrpl --version
```

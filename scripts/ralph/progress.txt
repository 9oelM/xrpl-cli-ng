## Codebase Patterns
- Vault commands MUST use devnet (wss://s.devnet.rippletest.net:51233) — SingleAssetVault amendment not on testnet
- Devnet faucet: https://faucet.devnet.rippletest.net/accounts — helper at tests/helpers/devnet.ts
- VaultCreate requires Fee = 200000 drops minimum (0.2 XRP) — set explicitly after autofill
- VaultID is extracted from CreatedNode.LedgerIndex where LedgerEntryType === 'Vault' in tx metadata
- All 6 vault subcommands (create/set/deposit/withdraw/delete/clawback) live in src/commands/vault.ts
- Asset parsing for vault commands: reuse parseAmount from utils/amount.ts, map xrp→{currency:'XRP'}, iou→{currency,issuer}, mpt→{mpt_issuance_id}
- For vault network tests: one faucet call in beforeAll, reuse same wallet seed for all tests (vaults are per-wallet independent)

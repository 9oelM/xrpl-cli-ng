## Codebase Patterns
- Clawback IOU wire format: Amount.issuer = holder address (not the token issuer)
- Clawback MPT wire format: Amount = { value, mpt_issuance_id }, Holder = holder address
- Commander --no-wait: use `options.wait` (boolean), no default arg; check `if (!options.wait)` for no-wait path
- MPTokenIssuanceCreateFlags.tfMPTCanClawback must be set at issuance time for MPT clawback to work
- Use AccountSetAsfFlags.asfAllowTrustLineClawback (value 16) to enable IOU clawback on issuer account
